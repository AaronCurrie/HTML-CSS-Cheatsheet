HTML and CSS Cheatsheet

Semantic HTML

Tag: main

Main

This is the main element of the page where all other elements are placed within

Tag: section

Section

The section element defines a section in a document. A section is a thematic grouping of content, typically with a heading.

Tag: article

Article

The article element specifies independent, self-contained content. An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site.

Tag: header

Header

The header element represents a container for introductory content or a set of navigational links.

Tag: footer

Footer

The footer element defines a footer for a document or section.

Tag: aside

Aside

The aside element defines some content aside from the content it is placed in (like a sidebar). The aside content should be indirectly related to the surrounding content.

Tag: figure

Figure

The figure tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.

Tag: figcaption

Fig Caption

The figcaption tag defines a caption for a figure element. The figcaption element can be placed as the first or as the last child of a figure element.

HTML Table References

Tag: table

Table

The wrapper element for all HTML tables.

Tag: thead

Table Head

The set of rows defining the column headers in a table.

Tag: tbody

Table Body

The set of rows containing actual table data.

Tag: tr

Table Row

The table row container.

Tag: td

Table Data

The table row container.

Tag: tfoot

Table Foot

The set of rows defining the footer in a table.

Attribute: colspan

Column Span

Defines how many columns a td element should span.

Attribute: rowspan

Row Span

Defines how many rows a td element should span.
